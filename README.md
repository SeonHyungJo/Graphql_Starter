# Graphql 장점

- Under-fetching
내가 원하는 하나의 요청을 성공적으로 하기 위한 다른 요청을 하는 것. 예를 들어 하나의 화면을 완성하기 위해서 여러 번의 요청을 하는 것...

> 하나의 예로 위메프에서 잠깐 있었을때 ajax로 메인화면을 호출할 때 3번을 요청하는 것을 말하는 것

- Over-fetching
내가 요청한 정보보다 더 많은 정보를 서버로 부터 받게 되는 것
